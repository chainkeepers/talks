===================================

Lektoři: Ondřej Vostal (www.chainkeepers.io, 0xkron)

Na co se můžete těšit
=====================

A. Posílání arbitrážních transakcí
++++++++++++++++++++++++++++++++++

- z čeho se skládá zisková arbitráž
